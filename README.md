# 表达式计算器 (ExprEval_cpp)

一个用 C++ 编写的轻量级数学表达式求值器。
![C++](https://img.shields.io/badge/C++-17-blue)

## 功能特性

- 支持运算符：`+`, `-`, `*`, `/`, `%`, `^`, `(`, `)`
- 处理一元负号（负数）
- 支持整数和浮点数运算
- 运算符优先级和结合性处理
- 中缀表达式转后缀表达式
- 错误处理（除零错误、表达式格式错误等）
